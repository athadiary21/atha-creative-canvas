Sintaks Dasar Media Queries

Struktur Media Queries
@media [media-type] and (media-feature) {
  /* CSS Rules */
}
looks_one
@media - Aturan CSS untuk memulai media query
looks_two
media-type - Jenis media (screen, print, all)
looks_3
media-feature - Karakteristik perangkat (width, height, orientation)
looks_4
operator - Kondisi (and, not, only)

Media Types

Jenis Media yang Dapat Ditargetkan
devices
all
Semua perangkat (default)
computer

Perangkat dengan layar (komputer, tablet, ponsel)

print
Mode pratinjau cetak atau printer
record_voice_over
speech
Pembaca layar (screen reader)
/* Target semua perangkat */
@media all { /* CSS rules */ }

/* Target layar saja */
@media screen { /* CSS rules */ }

/* Target mode cetak */
@media print { /* CSS rules */ }
