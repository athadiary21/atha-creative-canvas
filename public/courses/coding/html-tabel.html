Tag dan Atribut untuk Setiap Jenis List

format_list_bulleted

Unordered List

<ul> <li>
arrow_right
type: jenis bullet (disc, square, circle)
<ul type="square">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
format_list_numbered

Ordered List

<ol> <li>
arrow_right
type: jenis penomoran (1, A, a, I, i)
arrow_right
start: nomor awal
<ol type="A" start="3">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
description

Description List

<dl> <dt> <dd>
arrow_right
dt: istilah yang didefinisikan
arrow_right
dd: deskripsi dari istilah
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>

Contoh Penggunaan List dengan Nesting

code

Kode HTML

<ul>
  <li>Buah-buahan</li>
  <li>Minuman</li>
  <li>Makanan</li>
    <ul>
      <li>Nasi Goreng</li>
      <li>Mie Ayam</li>
      <li>Soto</li>
    </ul>
  <li>Cemilan</li>
</ul>
lightbulb
List dapat disarangkan di dalam list lain untuk membuat struktur hierarki. Indentasi kode membantu meningkatkan keterbacaan.
visibility

Hasil Tampilan

  • Buah-buahan
  • Minuman
  • Makanan
    • Nasi Goreng
    • Mie Ayam
    • Soto
  • Cemilan
info
Browser secara otomatis akan menampilkan list bersarang dengan indentasi dan bullet yang berbeda untuk membedakan tingkat hierarki.

Tag dan Atribut untuk Tabel

table_chart

Elemen Struktur Tabel

<table> <tr> <td> <th>
arrow_right
<table>: Membungkus seluruh tabel
arrow_right
<tr>: Mendefinisikan baris tabel (table row)
arrow_right
<td>: Mendefinisikan sel data (table data)
arrow_right
<th>: Mendefinisikan header sel (teks tebal & tengah)
merge_type

Atribut Spanning

arrow_right
colspan: Menggabungkan beberapa kolom
arrow_right
rowspan: Menggabungkan beberapa baris
code

Contoh Kode Lengkap

<table>
  <caption>Data Penjualan</caption>
  <thead>
    <tr>
      <th>Produk</th>
      <th>Harga</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>Rp 10jt</td>
    </tr>
  </tbody>
</table>
visibility

Hasil
