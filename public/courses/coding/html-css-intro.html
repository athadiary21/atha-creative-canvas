Apa itu HTML?

code Definisi

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web.

build Fungsi

HTML digunakan untuk membuat struktur dan konten dasar halaman web.

architecture Struktur Dasar

Struktur HTML terdiri dari DOCTYPE, html, head, dan body.

Elemen-elemen HTML

code Tag HTML

Tag terdiri dari tag pembuka dan tag penutup yang mengelilingi konten.

settings Atribut HTML

Atribut memberikan informasi tambahan tentang elemen:

href src class id

view_quilt Elemen Umum

head body p h1-h6 div span a img ul li
<!DOCTYPE html>
<html>
  <head>
    <title>Contoh Halaman</title>
  </head>
  <body>
    <h1 id="judul">Judul Halaman</h1>
    <p class="paragraf">Ini adalah paragraf.</p>
    <a href="https://example.com">Link</a>
    <img src="gambar.jpg" alt="Deskripsi">
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
    </ul>
  </body>
</html>

Apa itu CSS?

brush Definisi

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format dokumen HTML.

style Fungsi

CSS digunakan untuk mengatur tampilan halaman web, termasuk warna, font, layout, dan animasi.
