CSS3: Konsep Box Model
(Margin, Padding, Border)

Memahami Struktur Elemen dalam CSS

Apa itu CSS Box Model?

infoDefinisi
  • Model yang membungkus elemen HTML di browser
  • Setiap elemen dianggap sebagai kotak (box)
  • Memudahkan pengaturan ukuran, posisi, dan properti elemen
starPentingnya Box Model
  • Dasar untuk membuat layout yang rapi
  • Mengatur jarak antar elemen
  • Kontrol penuh terhadap tampilan visual website

Komponen Box Model: Content

articleApa itu Content?
  • Area inti dari sebuah box
  • Tempat konten aktual ditampilkan
  • Berupa teks, gambar, atau elemen lain
straightenMengatur Ukuran Content
  • Properti width untuk lebar
  • Properti height untuk tinggi
  • Dapat menggunakan berbagai unit (px, %, em, dll)
/* Mengatur ukuran content */
width: 300px;
height: 200px;

Komponen Box Model: Padding

space_barApa itu Padding?
  • Ruang di dalam border di sekitar konten
  • Menciptakan jarak antara konten dan border
  • Properti transparan (tidak memiliki warna)
codeProperti Padding
  • padding-top - jarak di atas
  • padding-right - jarak di kanan
  • padding-bottom - jarak di bawah
  • padding-left - jarak di kiri
/* Individual properties */
padding-top: 20px;
padding-right: 15px;
padding-bottom: 20px;
padding-left: 15px;
