What is useState Hook?

useState adalah Hook bawaan React yang memungkinkan komponen fungsional memiliki state.

Sebelum React 16.8, hanya komponen kelas yang bisa memiliki state. Dengan useState, kita bisa menambahkan state ke komponen fungsional tanpa perlu mengubahnya menjadi kelas.

check_circleMenyimpan data yang bisa berubah seiring waktu
check_circleMemicu re-render komponen saat state berubah
check_circleMengembalikan array dengan dua elemen: nilai state dan fungsi untuk mengubahnya
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

State tidak boleh diubah langsung (mutasi). Selalu gunakan fungsi setter.

const increment = () => {
  setCount(count + 1);
}

const incrementByFive = () => {
  setCount(prev => prev + 5);
}

Objects & Arrays

Prinsip Immutability

content_copyBuat salinan baru, jangan mutasi
filter_noneGunakan spread operator (...)
lightbulbMengapa?

React menggunakan referensi objek untuk mendeteksi perubahan.

// Object
setUser(prev => ({ ...prev, name: 'New' }));

// Array Add
setItems(prev => [...prev, newItem]);

// Array Remove
setItems(prev => prev.filter(i => i.id !== id));
