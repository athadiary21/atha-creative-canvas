PreventDefault & StopPropagation

blockpreventDefault()

Menghentikan aksi default browser (misal: submit form, link).

form.addEventListener('submit', e => {
  e.preventDefault();
});

cancelstopPropagation()

Menghentikan propagasi event ke elemen lain.

child.addEventListener('click', e => {
  e.stopPropagation();
});
MetodeFungsi
preventDefaultStop aksi browser
stopPropagationStop bubbling/capturing

codeContoh Praktis

  • check_circleValidasi Form (preventDefault)
  • check_circleDropdown Menu (stopPropagation)
