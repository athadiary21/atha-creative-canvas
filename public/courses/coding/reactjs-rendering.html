Pengenalan Conditional Rendering

Conditional Rendering adalah proses menampilkan UI yang berbeda berdasarkan kondisi tertentu dalam aplikasi React.

Menggunakan logika JavaScript untuk menentukan elemen mana yang akan dirender

Menampilkan tombol "Login" jika pengguna belum masuk, atau "Logout" jika sudah masuk

Metode Conditional Rendering (Bagian 1)

code

Pernyataan If/Else

Menggunakan pernyataan JavaScript standar untuk mengembalikan JSX yang berbeda berdasarkan kondisi.

function UserGreeting(props) {
  if (props.isLoggedIn) {
    return <h1>Selamat datang kembali!</h1>;
  }
  return <h1>Silakan masuk terlebih dahulu</h1>;
}
lightbulbKapan Menggunakan

Untuk logika yang kompleks atau ketika perlu mengeksekusi kode tambahan sebelum mengembalikan JSX

compare_arrows

Operator Ternary

Sintaks ringkas untuk menulis ekspresi bersyarat dengan format: kondisi ? nilai_jika_benar : nilai_jika_salah

function UserGreeting(props) {
  return (
    <div>
      {props.isLoggedIn ? 
        <h1>Selamat datang kembali!</h1> : 
        <h1>Silakan masuk terlebih dahulu</h1>
      }
    </div>
  );
}
lightbulbKapan Menggunakan

Untuk kondisi sederhana dengan dua kemungkinan hasil, membuat kode lebih ringkas dan mudah dibaca

Metode Conditional Rendering (Bagian 2)

add

Operator Logis AND (&&)

Menampilkan elemen hanya jika kondisi bernilai true, tanpa alternatif jika false.

function Notification(props) {
  return (
    <div>
      {props.hasMessage && 
        <div>Anda memiliki pesan baru</div>
      }
    </div>
  );
}
lightbulbKapan Menggunakan

Untuk menampilkan elemen secara opsional tanpa alternatif jika kondisi false

Perhatian: Hindari angka di sisi kiri && (gunakan messageCount > 0 && bukan messageCount &&)

assignment

Variabel Kondisional

Menetapkan JSX ke variabel berdasarkan kondisi, lalu merender variabel tersebut.

function UserStatus(props) {
  let statusElement;
  if (props.isLoggedIn) {
    statusElement = <span>Aktif</span>;
  } else {
    statusElement = <span>Tidak Aktif</span>;
  }
  return <div>Status: {statusElement}</div>;
}
lightbulbKapan Menggunakan

Untuk logika yang kompleks atau ketika perlu merender elemen yang sama dengan variasi kecil. Cara paling fleksibel.

Contoh Praktis & Praktik Terbaik

codeContoh Praktis: Login/Logout
import React, { useState } from "react";

const App = () => {
  const [isLoggedIn, setIsLoggedIn] = useState(false);
  
  const handleLogin = () => setIsLoggedIn(true);
  const handleLogout = () => setIsLoggedIn(false);
  
  return (
    <div>
      {isLoggedIn ? (
        <div>
          <h1>Selamat Datang Kembali!</h1>
          <button onClick={handleLogout}>Logout</button>
        </div>
      ) : (
        <div>
          <h1>Silakan Login</h1>
          <button onClick={handleLogin}>Login</button>
        </div>
      )}
    </div>
  );
};
