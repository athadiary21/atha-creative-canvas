Pengertian Fungsi dalam JavaScript

Fungsi adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu.

check_circle

Fungsi adalah first-class objects dalam JavaScript

check_circle

Dapat menerima input (parameter) dan mengembalikan output (return value)

check_circle

Memungkinkan kode lebih terorganisir, modular, dan efisien

check_circle

Dapat dipanggil berulang kali dengan nilai input yang berbeda

check_circle

Fungsi dapat disimpan dalam variabel, objek, atau array

Parameter dan Argumen

infoPerbedaan Parameter dan Argumen

Parameter adalah variabel yang didefinisikan dalam deklarasi fungsi

Argumen adalah nilai aktual yang diteruskan saat fungsi dipanggil

compare_arrowsParameter vs Argumen

settings

Parameter

• Didefinisikan di dalam fungsi
• Bersifat placeholder
• Digunakan dalam tubuh fungsi

input

Argumen

• Diberikan saat memanggil fungsi
• Merupakan nilai aktual
• Mengisi parameter

codeContoh Kode

// name adalah parameter
function greet(name) {
  console.log("Hello, " + name + "!");
}

// "JavaScript" adalah argumen
greet("JavaScript");

settings_input_componentDefault Parameter

Memberikan nilai default jika argumen tidak diberikan

more_horizRest Parameter

Mengumpulkan argumen menjadi array dengan ...

// Default Parameter
function greet(name = "Guest") {
  console.log("Hello, " + name);
}

// Rest Parameter
function sum(...numbers) {
  return numbers.reduce((a, b) => a + b, 0);
}

Return Value

arrow_backKonsep Return Value

Return value adalah nilai yang dikembalikan oleh fungsi setelah selesai dieksekusi

Fungsi
arrow_forward
Return Value
check_circle

Ditentukan dengan keyword return

check_circle

Fungsi berhenti saat return dieksekusi

check_circle

Default: undefined jika tidak ada return

data_objectTipe Data Primitif

String, Number, Boolean, null, undefined

categoryTipe Data Kompleks

Object, Array, Function, Date, RegExp

codeContoh Return Value

// Fungsi dengan return value
function add(a, b) {
  return a + b;
}

// Menyimpan return value ke variabel
let result = add(5, 10);
console.log(result); // 15
// Fungsi tanpa return (undefined)
function greet(name) {
  console.log("Hello, " + name);
}

let greeting = greet("JavaScript");
console.log(greeting); // undefined
// Return value berupa objek
function createPerson(name, age) {
  return {
    name: name,
    age: age,
    greet: function() {
      return "Hi, I'm " + name;
    }
  };
}

Jenis-jenis Fungsi dalam JavaScript

label

Named Function

Fungsi dengan nama yang dapat dipanggil kembali

function greet() {
  return "Hello!";
}
help_outline

Anonymous Function

Fungsi tanpa nama, biasanya disimpan dalam variabel

const greet = function() {
  return "Hi there!";
};
arrow_forward

Arrow Function

Sintaks singkat fungsi dengan =>

const square = n => n * n;
console.log(square(4)); // 16
play_arrow

IIFE

Immediately Invoked Function Expression - fungsi yang langsung dieksekusi

(function() {
  console.log("Runs immediately!");
})();
sync

Callback Function

Fungsi yang diteruskan sebagai argumen ke fungsi lain

function process(n, callback) {
  return callback(n);
}
repeat

Recursive Function

Fungsi yang memanggil dirinya sendiri

function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}

Best Practices dalam Menggunakan Fungsi

Tips dan praktik terbaik untuk menulis fungsi JavaScript yang efektif dan efisien

title

Nama yang Deskriptif

Gunakan nama fungsi yang jelas menggambarkan apa yang dilakukan
// Baik
function calculateTotalPrice(items) { ... }

// Kurang Baik
function x(items) { ... }
text_format

Satu Tugas Saja

Setiap fungsi seharusnya melakukan satu tugas dengan baik
// Baik
function validateEmail(email) { ... }
function sendEmail(email, message) { ... }
short_text

Pendek dan Sederhana

Pertahankan fungsi singkat dan mudah dibaca
// Fungsi dengan kurang dari 20-30 baris
// Lebih mudah dipahami dan diuji
repeat_one

Hindari Duplikasi

Gunakan fungsi untuk menghindari kode duplikat
// Jangan ulang kode yang sama
// Ekstrak ke fungsi yang dapat digunakan kembali
comment

Dokumentasi

Tambahkan komentar yang jelas untuk fungsi kompleks
/**
* Menghitung total harga
* @param {Array} items - Daftar item
* @returns {Number} Total harga
*/
verified

Fungsi Murni

Prioritaskan pure functions yang tidak memiliki side effects
// Input yang sama → Output yang sama
// Tidak mengubah data di luar fungsi
