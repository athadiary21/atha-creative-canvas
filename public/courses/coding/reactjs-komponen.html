Mengirim Props dari Induk ke Anak

arrow_forward

Direct Prop Passing

Mengirim props langsung dari komponen induk ke anak

<ChildComponent name="John" age={25} />
open_in_full

Spread Operator

Menggunakan spread operator untuk mengirim semua properti objek

const user = { name: "John", age: 25 };
<ChildComponent {...user} />
unpack

Destructuring

Mengekstrak props langsung di parameter fungsi

function ChildComponent({ name, age }) {
  return <div>{name}, {age}</div>;
}
functions

Passing Functions

Mengirim fungsi sebagai props untuk komunikasi dari anak ke induk

<ChildComponent handleClick={this.handleButtonClick} />
