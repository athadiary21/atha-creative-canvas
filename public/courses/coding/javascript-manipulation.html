Pengenalan DOM

Apa itu DOM?

Document Object Model (DOM) adalah representasi struktur HTML/XML dalam bentuk objek yang dapat dimanipulasi dengan JavaScript.

Mengapa DOM Penting?
  • check_circle Memungkinkan JavaScript mengakses dan mengubah konten HTML
  • check_circle Membuat halaman web menjadi interaktif dan dinamis
  • check_circle Memungkinkan pembaruan konten tanpa reload halaman
  • check_circle Dasar dari hampir semua aplikasi web modern

Struktur DOM

Jenis-jenis Node
codeElement Nodes

Representasi elemen HTML seperti <div>, <p>, <img>

Konten teks yang ada di dalam elemen HTML

Atribut HTML seperti id, class, src, href

Komentar dalam kode HTML

Struktur Pohon (Tree)
  • account_tree Dimulai dari root node (document)

  • compare_arrows Node-node pada level yang sama disebut siblings

Metode Akses Elemen DOM

tag
getElementById()

Mengambil elemen berdasarkan ID-nya

const el = document.getElementById("myId");
style
getElementsByClassName()

Mengambil semua elemen dengan kelas tertentu

const els = document.getElementsByClassName("cls");
search
querySelector()

Mengambil elemen pertama yang sesuai selector CSS

const el = document.querySelector(".cls #id");
find_in_page
querySelectorAll()

Mengambil semua elemen sesuai selector CSS

const els = document.querySelectorAll("div");
lightbulb querySelector() dan querySelectorAll() lebih fleksibel karena mendukung semua selector CSS
