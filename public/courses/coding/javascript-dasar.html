Diciptakan oleh Brendan Eich pada tahun 1995 di Netscape.

Variabel dalam JavaScript

storageApa itu Variabel?
Tempat untuk menyimpan data di dalam memori komputer.
codeCara Mendeklarasikan
var nama = "John"; // Lama
let umur = 25; // Modern
const pi = 3.14; // Konstanta
ruleAturan Penamaan
  • check_circleDimulai dengan huruf, underscore (_), atau dollar ($)
  • check_circleTidak boleh ada spasi
  • check_circleCase-sensitive: nama ≠ Nama
  • check_circleGunakan camelCase
lightbulbBest Practices
• Gunakan const jika nilai tetap
• Gunakan let jika nilai berubah
• Hindari var
