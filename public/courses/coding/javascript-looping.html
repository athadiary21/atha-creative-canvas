Perulangan For

Struktur dan implementasi perulangan for dalam JavaScript

codeSintaks Dasar
for (inisialisasi; kondisi; ekspresi akhir) {
  // blok kode yang akan diulang
}
Inisialisasi
Dieksekusi sekali di awal perulangan
Kondisi
Dievaluasi sebelum setiap iterasi
Ekspresi Akhir
Dieksekusi setelah setiap iterasi
play_arrowContoh Penggunaan
// Perulangan ascending
for (let i = 0; i < 5; i++) {
  console.log(`Iterasi ke-${i}`);
}

const buah = ['Apel', 'Mangga', 'Pisang'];

for (let i = 0; i < buah.length; i++) {
  console.log(buah[i]);
}

Perulangan While

Struktur dan implementasi perulangan while dalam JavaScript

codeSintaks Dasar
while (kondisi) {
  // blok kode yang akan diulang
}
Kondisi
Dievaluasi sebelum setiap iterasi. Jika true, blok kode dieksekusi
play_arrowContoh Penggunaan
// Counter dengan while
let count = 1;
while (count <= 5) {
  console.log(`Count: ${count}`);
  count++;
}
compare_arrowsKarakteristik While Loop

Kondisi Awal
Kondisi diperiksa sebelum eksekusi blok kode

Update Manual
Perlu update variabel kondisi di dalam loop

Kontrol Perulangan

Mengontrol alur eksekusi perulangan dengan break dan continue

stop

Break

Menghentikan perulangan secara paksa

Keluar loop
Kondisi khusus
codeContoh Break
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}
skip_next

Continue

Melanjutkan ke iterasi berikutnya

Lewati iterasi
Filter data
codeContoh Continue
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i);
}
